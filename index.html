01 / Overview

What is
RDKit?

RDKit (Reaction Design Toolkit) is a collection of cheminformatics and machine-learning tools written in C++ and Python. It was developed by Greg Landrum and is now widely used in the pharmaceutical, biotech, and academic communities.

At its core, RDKit allows chemists and computer scientists to work with molecular structures programmatically. You can read and write standard chemical file formats like SMILES, SDF, and Mol2, perform substructure searches, calculate molecular fingerprints for similarity analysis, and generate 2D/3D molecular depictions.

RDKit is tightly integrated with Python's scientific ecosystem — NumPy, Pandas, scikit-learn — making it a powerful foundation for machine learning in chemistry and AI-driven drug discovery.

02 / Capabilities

Core Features

🔬

Molecule I/O

Read and write SMILES, SMARTS, SDF, Mol2, PDB, and InChI formats. Interconvert between all major representations used in cheminformatics workflows.

📐

2D/3D Depiction

Generate publication-quality 2D drawings with the MolDraw2D engine. Compute 3D coordinates with ETKDG conformer generation and UFF/MMFF force fields.

🧮

Property Calculation

Calculate molecular weight, logP, TPSA, hydrogen bond donors/acceptors, rotatable bonds, ring count, and 200+ RDKit descriptors instantly.

🔍

Substructure Search

SMARTS-based substructure matching. Atom mapping, ring detection, aromatic system perception, and MCS (maximum common substructure) algorithms.

🧬

Fingerprints & Similarity

Morgan (ECFP), Topological Torsion, MACCS Keys, RDKit, and Avalon fingerprints. Tanimoto, Dice, and cosine similarity metrics for virtual screening.

⚗️

Chemical Reactions

Reaction SMARTS for applying synthetic transformations. Retrosynthetic analysis tools, reaction enumeration, and combinatorial library generation.

04 / Reference

SMILES Notation
Quick Reference

C
Carbon
Aliphatic carbon atom. CC = ethane, CCC = propane.
c
Aromatic Carbon
Lowercase = aromatic. c1ccccc1 = benzene ring.
N
Nitrogen
Aliphatic nitrogen. CN = methylamine (trimethyl with 3 C's).
O
Oxygen
CO = methanol, CCO = ethanol, C=O = carbonyl.
=
Double Bond
C=C = ethylene. C=O = aldehyde/ketone/carboxyl.
#
Triple Bond
C#N = nitrile. C#C = acetylene.
( )
Branches
CC(C)C = isobutane. Parentheses denote side chains.
1..9
Ring Closure
C1CCCCC1 = cyclohexane. Numbers pair ring open/close atoms.
05 / Real-World Use

Where RDKit
Is Used

01

Drug Discovery

Pharmaceutical companies use RDKit to screen millions of virtual compounds (virtual HTS), compute ADMET properties, filter by Lipinski's Rule of Five, and cluster compound libraries for lead optimization.

02

Machine Learning in Chemistry

RDKit generates molecular fingerprints and descriptors used as features for ML models. Combined with scikit-learn or deep learning frameworks, it powers QSAR/QSPR predictive modeling.

03

Chemical Database Management

The PostgreSQL cartridge allows SQL queries on chemical structures — searching databases of millions of compounds by similarity, substructure, or exact match at database speed.

04

Materials Science & Green Chemistry

Researchers use RDKit to design novel polymers, optimize reaction conditions, predict solubility, and identify sustainable synthetic pathways with minimal waste.
